(async) load(name, baseURL, ext)

Genera una etiqueta script que carga el archivo desde la url. La url base por defecto es relativa al archivo HTML que llama a los componentes, sin embargo, se puede mandar a llamar cualquier CDN que almacene el script. Si la url ya ha sido consultada previamente será ignorada.
Parameters:
Name Type Default Description
name string Nombre físico del componente (nombre del archivo o ruta)
baseURL string ./ Url de la ruta anterios al nombre del archivo
ext string .js Extensión del archivo

Examples

Cargar un script local

run(async () => {
 await load("example/Button");
 const button = Button();
 document.appendChild(button);
})

Cargar un script remoto

run(async () => {
 await load("https://cdn.jsdelivr.net/gh/badillosoft/mutator@master/example/Button");
 const button = Button();
 document.appendChild(button);
})

mutate(element) → {function}

Aplicador Muta un elemento cualquiera mediante mutadores. La función encapsula un elemento y devuelve una segunda función que recibe en sus parámetros mutadores (funciones de mutación) y se las aplica al elemento encapsulado.
Parameters:
Name Type Description
element HTMLElement | string | any Elemento a mutar mediante los mutadores

Examples

Sintaxis

// mutate(element)(
//   mutator_1(...config_1),
//   mutator_2(...config_2),
//   ...,
//   mutator_N(...config_N),
//);
mutate("button")(
 domAttribute({ innerText: "click here!" }),
 domStyle({ color: "blue" }),
 domEventCallback("click", () => alert("hi")),
);

Crear un aplicador a partir de un string sin mutadores

const button = mutate("button")();
document.body.appendChild(button);

Crear un aplicador a partir de un string con mutadores

const button = mutate("button")(
 domStyle({ color: "cornflowerblue" }),
 domAttribute({ innerText: "Hello world" }),
);
document.body.appendChild(button);

Crear una función que devuelva un aplicador configurado

const Button = (text, color) => mutate("button")(
 domStyle({ color }),
 domAttribute({ innerText: text }),
);
const button = Button("hello world", "crimson");
document.body.appendChild(button);

Aplicar mutadores a un elemento DOM existente

const button = document.createElement("button");
mutate(button)(
 domAttribute({ innerText: "hey you!" });
);
document.body.appendChild(button);
